Wordlist dictionary generator

Beginners learning brute-forcing attacks against WPA handshakes are often let down by the limitations of default wordlists like rockyou based on stolen passwords. The science of brute-forcing goes beyond using these default lists, allowing us to be more efficient by making customized wordlists. The most basic brute force attack is a dictionary attack, where the attacker works through a dictionary of possible passwords and tries them all.

This is why I've created worldlist dictionary generator, to help you make you own worldlist based on what you know about the target.
